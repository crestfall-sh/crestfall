## Crestfall Studio

### Notes

#### Production Build for Crestfall Core

```sh
cd ./crestfall-studio/
npm install
npm run studio-build
```

#### Development Build

```sh
cd ./crestfall-studio/
npm install
npm run studio-dev
```

#### Hard Links

```sh
ln ./package.json ./crestfall-studio/package.json
ln ./package-lock.json ./crestfall-studio/package-lock.json
```
